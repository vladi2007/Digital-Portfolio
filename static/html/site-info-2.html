Почему стоит пользоваться цифровым портфолио?

  • занимает меньше места, по сравнению с бумажным вариантом

  • простой интерфейс, как для просмотра, так и для введения

  • быстрый доступ, при наличии ссылки можно быстро ознакомиться с вашим портфолио

  • Отсутсвует риск повреждения и потери наград
